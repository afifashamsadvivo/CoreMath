# CoreMath

CoreMath is a JavaScript library with CSS styling for mathematical computations and visualizations.

## Features

- **JavaScript Core**: CoreMath.js provides the mathematical functionality
- **CSS Styling**: CoreMathPaintings.css for elegant styling and design

## Language Composition

- **JavaScript**: 61.8%
- **CSS**: 38.2%

## Description

CoreMath.js AND CSS for style (CoreMathPaintings.css, if possible)

## Getting Started

To use CoreMath, simply include the CoreMath.js file in your project and apply the CoreMathPaintings.css stylesheet for styling.
